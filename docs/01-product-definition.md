# 1. PRODUCT DEFINITION

**One sentence:**
Vitruvian is an AI-guided assisted stretching machine that uses motorized cable resistance, real-time sensors, and adaptive software to deliver professional-grade stretching sessions without a human therapist.

**One paragraph:**
Vitruvian is a standing/seated stretching station — roughly the size of a cable crossover machine — where an athlete steps in, attaches padded limb cuffs connected to motorized cables, and follows a guided stretching routine on a touchscreen. Force sensors and a depth camera track the user's range of motion in real time. The software adjusts cable tension and stretch targets based on how the user's body is actually responding — not a generic preset. Think of it as a StretchLab therapist replaced by a machine that never forgets your limits, never pushes too hard, and is available 24/7. It belongs in an NFL training room, a physical therapy clinic, or a high-end gym — not a sci-fi movie.
