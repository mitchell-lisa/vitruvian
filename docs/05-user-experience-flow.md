# 5. USER EXPERIENCE FLOW

## Full Session Walkthrough (~15 minutes)

**STEP 1 — APPROACH (0:00)**
User walks up to the machine. It's open-framed, well-lit, looks like premium gym equipment. A subtle LED strip on the pillars pulses slowly to indicate it's idle and ready. The touchscreen shows: **"Step in to begin."**

**STEP 2 — IDENTIFY (0:10)**
User steps onto the platform. The screen prompts:
- **Returning user:** "Welcome back, Marcus. Tap to start." (Recognized via phone NFC tap, QR code, or 4-digit PIN.)
- **New user:** "New here? Let's set you up. Takes 2 minutes." (Enter name, height, weight, any injuries/restrictions.)

**STEP 3 — SELECT ROUTINE (0:30)**
The screen shows today's recommended routine based on the user's history:
> **"Suggested: Lower Body Recovery (12 min)"**
> *Hamstrings, hip flexors, quads, calves*
> *Last session: 2 days ago — Upper Body*

User can accept the suggestion or browse other routines:
- Full Body (20 min)
- Upper Body (12 min)
- Lower Body (12 min)
- Pre-Workout Activation (8 min)
- Post-Workout Recovery (15 min)
- Custom (pick individual stretches)

**STEP 4 — SETUP (1:00)**
Screen shows clear visual instructions:
> **"Attach ankle cuffs to both legs. Clip to the LOW cable hooks."**
> *(Animated diagram showing exactly how)*

User grabs the appropriate cuffs from hooks on the pillar, straps them on (velcro + buckle), and clips the carabiner to the cable. The machine applies a tiny 1 lb "test pull" on each cable to confirm the connection. Screen shows green checkmarks: **"Left ankle — connected. Right ankle — connected."**

If the routine also requires the hip belt or wrist cuffs, the screen guides those next.

**STEP 5 — BASELINE SCAN (1:30)**
> **"Stand naturally. Arms at your sides. Hold still for 5 seconds."**

The depth camera captures the user's resting posture. The system notes:
- Shoulder height symmetry
- Hip alignment
- Standing posture baseline

This takes 5 seconds. The user barely notices it.

**STEP 6 — STRETCHING BEGINS (2:00)**

The screen shows the first stretch with a clear visual:
> **"LEFT HAMSTRING STRETCH"**
> *Lift your left leg forward. The machine will gently pull.*
> **[3... 2... 1... Starting]**

The cable engages. Tension builds slowly (0.5 lb/sec). The screen shows:
- A real-time ROM gauge (like a speedometer): current angle vs. target angle
- Current tension level (low / medium)
- A timer counting down the hold (30 seconds)
- A big **"TOO MUCH"** button if the user wants to back off

During the hold:
- The system maintains steady tension
- If ROM increases (user relaxes deeper), tension adjusts to follow
- If the user tenses up, tension reduces immediately
- The screen may show: **"Relax your shoulders"** or **"Breathe out slowly"** based on camera observation

**STEP 7 — TRANSITION BETWEEN STRETCHES (2:45)**
After the hold timer ends, cables slowly release tension over 3 seconds. Screen shows:
> **"Nice. Left hamstring: 78 degrees today (up 2 degrees from last session)"**
> **"Switching sides. Right hamstring next."**
> **[10 second rest]**

User repositions if needed. The next stretch begins.

**STEP 8 — MID-SESSION ADJUSTMENT (7:00)**
Halfway through the routine, the system may adapt:
- "Your left hip flexor is tighter than usual today. Adding an extra 15-second hold."
- "Your right shoulder ROM hit a new high — great progress."
- If user has been hitting "TOO MUCH" frequently: "Reducing intensity for the rest of this session."

These adjustments happen automatically but are communicated clearly on screen.

**STEP 9 — SESSION ENDS (14:00)**
Final stretch completes. All cables go slack. Screen shows:

> **SESSION COMPLETE**
> **Duration:** 12:24
> **Stretches:** 8
> **Notable:**
> - Left hamstring: +2 degrees (new personal best)
> - Hip flexors: holding steady
> - Right shoulder: +3 degrees improvement this week
>
> **"Remove cuffs and hang them on the hooks. See you next time."**

**STEP 10 — EXIT**
User unclips cuffs, hangs them up, steps off the platform. The machine returns to idle state. LED strip pulses slowly again.

If the user has the companion app, the session summary syncs automatically via WiFi.

## Session Timing Summary

| Phase | Duration |
|-------|----------|
| Approach + ID | ~30 sec |
| Routine selection | ~30 sec |
| Cuff setup | ~60 sec |
| Baseline scan | ~5 sec |
| Active stretching | 8-20 min |
| Cooldown + summary | ~30 sec |
| **Total wall time** | **~12-24 min** |
